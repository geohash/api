    Introduction

    Welcome to the Geoha.sh. You can use our API to generate geohashes at arbitrary precisions.

    Geohash

    Generate a Geohash

    curl "http://api.geoha.sh/q?longitude=123&latitude=456&precision=1"

    The above command returns JSON structured like this:

    {
    "geohash": "s00"
}

    HTTP Request

    `GET http://api.geoha.sh/q

    Query Parameters

    Parameter Default Description
    latitude None. The latitude of the location you are geohashing.
    longitude None. The longitude of the location you are geohashing.
    precision None. The precision of the location you are geohashing.
